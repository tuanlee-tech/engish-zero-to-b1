# Prompt: Chữa lỗi & Giải thích ngữ pháp

## Chữa bài viết

```
Chữa đoạn văn sau của tôi:

[DÁN BÀI VIẾT VÀO ĐÂY]

Yêu cầu:
1. Liệt kê từng lỗi: [sai] → [đúng] + giải thích tiếng Việt
2. Đưa bản hoàn chỉnh đã sửa
3. Nếu lỗi lặp lại từ weak_spots.md → nhắc nhở đặc biệt
4. Tạo 1 câu tương tự để tôi luyện lại điểm vừa sai
5. Cập nhật memory/weak_spots.md nếu có lỗi mới
```

## Giải thích ngữ pháp

```
Giải thích cho tôi về: [GRAMMAR POINT]

Format:
1. Giải thích bằng tiếng Việt, đơn giản nhất có thể
2. Công thức (dùng S/V/O/Adj...)
3. 3 ví dụ: câu đơn giản → câu phức tạp hơn
4. So sánh với tiếng Việt (điểm giống/khác)
5. Lỗi người Việt hay mắc với điểm này
6. 1 bài tập nhỏ để tôi tự thực hành
```
